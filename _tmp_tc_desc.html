1. Validation Scope

Validates the PCT cross-product interaction invariants defined in TCD 16031169419 §5: ordered throttle under RAPL PL1 (LP cores first), C-state topology stability (CLOS unchanged through C6), PROCHOT thermal override (all cores clamped), and EMTTM interaction (min of ordered and thermal limits).

Environment: NWP-IMH post-silicon FV. Requires PythonSV for TPMI register access and DTS thermal injection capability.

4. Test Steps

Phase A — RAPL Ordered Throttle

#ActionExpected Result
1Record baseline frequencies. For all 96 cores, read /sys/bus/cpu/devices/cpuN/cpufreq/scaling_cur_freq. Identify HP cores (CLOS[0]) and LP cores (CLOS[3]).HP cores at PCT TRL (~4.4 GHz); LP cores at LP clip (~P1). HP > LP for all core pairs.
2Start all-core stress workload. Launch stress-ng --cpu 96 --timeout 60s.All cores loaded. Baseline power near TDP.
3Reduce PL1 to 70% TDP. Write SOCKET_RAPL_PL1_CONTROL.PWR_LIM = 0.7 × TDP via PythonSV.RAPL PID loop begins throttling. Power converges toward PL1 target.
4Sample frequencies for 10 seconds. Poll scaling_cur_freq at 100 ms intervals for all cores. Compute HP-mean and LP-mean per sample.LP-mean < HP-mean for ≥ 90% of samples (ordered throttle: LP first).
5Check EXCURSION_TO_MIN. Read SST_CP_STATUS bits 11:8.EXCURSION_TO_MIN[3] = 1 (LP CLOS hit floor). EXCURSION_TO_MIN[0] may = 0 (HP maintained).
6Restore PL1 to TDP. Write original PL1 value back. Stop stress workload.Frequencies recover. Clear EXCURSION_TO_MIN via RESET_EXCURSION_TO_MIN.

Phase B — C-State Topology Stability

#ActionExpected Result
7Record PCT topology baseline. Run intel-speed-select perf-profile info -l 0. Save HP module count, APIC IDs.8 HP modules across both CBBs.
8Cycle HP cores through C6 1000 times. For each HP core, force idle (drop workload) → wait for C6 entry (poll cpuidle/state3/usage) → wake with short burst. Repeat 1000 cycles.C6 usage counter increments. No hangs, no MCEs.
9Verify topology unchanged. Re-run intel-speed-select perf-profile info -l 0. Compare against Step 7 baseline.HP module count and APIC IDs identical to baseline. No CLOS reassignment.
10Verify HP frequency on wake. After last C6 exit, immediately read HP core scaling_cur_freq.HP core frequency ≥ LP_CLIP + 100 MHz within 10 ms of wake (not clipped to LP level).

Phase C — Thermal Override

#ActionExpected Result
11Start all-core stress + reduce PL1 to 70% TDP. Same as Steps 2–3 to establish ordered throttle baseline.LP throttled first; HP maintained. Ordered throttle active.
12Inject PROCHOT. Via PythonSV DTS override, assert PROCHOT on socket 0. Record timestamp.All cores clamped to PROCHOT response frequency. HP and LP both at same low frequency.
13Verify PROCHOT override. Read scaling_cur_freq for all cores during PROCHOT.All cores ≤ PROCHOT response ratio. No core exceeds thermal ceiling.
14De-assert PROCHOT. Clear DTS override. Record timestamp.PCT frequency hierarchy resumes: HP > LP within 100 ms of de-assert.
15Restore PL1 and stop stress. Write original PL1. Kill workload.System returns to idle. No dmesg errors.
16Final state check. Read SST_CP_STATUS, IA32_THERM_STATUS, dmesg.No residual thermal events. EXCURSION_TO_MIN = 0x0 after clear. No MCEs or SST errors in dmesg.
5. Pass/Fail Measurement Method

Bar: Per TCD 16031169419 §5 Pass/Fail Bar.

Measurement procedure (this TC):

  1. Ordered throttle LP-first: LP-mean scaling_cur_freq < HP-mean for ≥ 90% of 100-sample window under PL1 = 70% TDP.
  2. EXCURSION_TO_MIN: SST_CP_STATUS[11:8] bit 3 = 1 under PL1 stress (LP CLOS at floor).
  3. C-state topology stable: intel-speed-select perf-profile info HP module count identical before/after 1000 C6 cycles.
  4. HP wake frequency: HP core scaling_cur_freq ≥ LP_CLIP + 100 MHz within 10 ms of C6 exit.
  5. PROCHOT override: All cores ≤ PROCHOT response ratio during assert.
  6. PROCHOT recovery: HP scaling_cur_freq > LP scaling_cur_freq within 100 ms of de-assert.
  7. Execution assertions: No MCEs, no SST/thermal dmesg errors. Test completes < 15 minutes.

FAIL if any numbered criterion fails. PROCHOT steps may be skipped if DTS injection unavailable (log skip reason).
