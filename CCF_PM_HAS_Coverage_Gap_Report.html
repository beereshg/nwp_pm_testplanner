Prioritized Gap Action Plan

Priority Flow ID Gap Title Recommended Action Why It Matters
P1 GV-08 AGV/FGV contention under C-state transitions Add to TCD 22022421168 (PEGA) or new TCD under TPF 22022420507 GV arbitration during C-state transitions is the core CCF PM correctness gate. A missed arbitration causes frequency glitch or system hang.
P1 IDLE-05 Ring C3 abort on mid-entry wake event Add to TCD 22022421179 (Idle States) Unhandled abort during Ring C3 entry causes in-flight transaction loss and potential deadlock.
P1 SCALE-08 PVP → IA_DISTRESS → CCF ring frequency New TC under TCD 22022421197 (Distress Signal) PVP throttle is the primary dV/dt protection path. If the PVP→distress chain breaks, silicon damage risk on power virus workloads.
P1 IDLE-07 LC6 / FC6 ring coordination New TCD under TPF 22022420512 (Idle States) LC6/FC6 are package-level C-states used in production. No coverage means entire power-off CCF sequence is unvalidated.
P2 UFF-01..07 All 7 Uniform Fabric Frequency TCs (TPF 22022420515) 7 existing TCDs (22022421207–22022421218) each need at least 1 TC Entire TPF 22022420515 has 0 TCs. Uniform fabric mode is a shipped feature used by VMM workloads — completely unvalidated.
P2 SCALE-09 ProcHot → ring frequency reduction New TC under TCD 22022421197 (Distress Signal) or new TCD ProcHot is a key thermal protection path. Ring frequency not scaling down on ProcHot causes thermal violation.
P2 IDLE-04 D2D → L1 transition during Ring C3 Extend TC 22022422873 (Ring C3) to check D2D link state Ring C3 without D2D L1 is not a complete ring drain. Leaving D2D in L0 during PLL shutdown causes UCIe link errors.
P2 GV-07 Per-die independent GV — NWP multi-die New TC under TCD 22022421171 (TPMI Interface) NWP-specific critical path. If per-die GV is broken, one CBB die can stall while another scales, breaking cache coherency latency.
P2 IDLE-09 Staggered module C6 entry ordering New TC under TCD 22022421179 (Idle States) didt from simultaneous module C6 entry can cause voltage droops. Stagger ordering validation prevents silicon reliability issues.
P3 GV-06 GVFSM state machine progression Add assertions to existing GV TCs or new TC under TPF 22022420507 GVFSM stuck in intermediate state causes CCF hang. State visibility helps debug future GV failures.
P3 SCALE-07 Distress fast path (100us DistressCycleUpdate) Extend TC 22022422905 (PCode algorithm) Fast path timing is important for ring frequency recovery under bursty LLC load. Slow path only validation misses timing SLA.
P3 IDLE-06 PKGC6 full entry/exit coordination (cNCU block/unblock) New TC under TCD 22022421179 (Idle States) or new TCD under TPF 22022420512 Ring C3 TC covers PLL shutdown but not cNCU block/unblock, which is the PC6 handshake with IMH. Incomplete PC6 path validation.
P3 BOOT-02 VCCRING SVID voltage management during GV Extend TC 22022422863 (VF Curves) to check voltage transitions Incorrect V-before-F or F-before-V sequencing during GV is a voltage drooping risk. VF curve TC only checks frequency endpoints.
P3 SCALE-10 Thermal throttling → ring frequency + VCCRING reduction New TC, possibly under a new TPF (CCF Thermal/Protection) Thermal-triggered ring scaling path is distinct from distress-based scaling. Complete thermal protection chain is unvalidated.
P3 IDLE-08 D2D active-idle state (NWP-specific) New TC under TPF 22022420512 (Idle States) NWP-only power-saving feature. Active-idle bugs manifest as unnecessary power consumption on compute chiplets.
