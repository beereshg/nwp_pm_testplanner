Core C-States › Fast C1E Env:
Validation Scope

Verify that Fast C1E control registers are correctly programmed on silicon after boot. This register validation confirms BIOS/firmware properly initialized the C1E subsystem and that the hardware is ready to execute C1E entry/exit flows.
