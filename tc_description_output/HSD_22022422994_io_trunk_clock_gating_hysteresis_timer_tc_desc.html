Validation Scope

Verify that PrimeCode correctly programs the **RC_MIO hysteresis timer** during PH6 per MAS §9.1. The hysteresis timer controls the delay between QCh idle detection and trunk clock gate assertion. Even though the timer never triggers on NWP (no L1), verifying the programmed value is correct ensures the infrastructure is ready for future silicon with L1 support.
