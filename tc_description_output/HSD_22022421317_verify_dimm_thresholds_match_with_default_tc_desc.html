Memory Thermal Management › MR4-based CLTT Env:
Validation Scope

Verify that BIOS correctly programs MC registers controlling DIMM thermal throttle thresholds (low/mid/high). These thresholds determine when MC applies bandwidth throttling based on MR4-reported DRAM temperatures. MR4-based CLTT uses per-DRAM die temperature sensors via JEDEC DDR5 Mode Register 4. MC reads MR4 periodically (~128 ms), and applies throttle/refresh actions when temp exceeds programmed thresholds.
