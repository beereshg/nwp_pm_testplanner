Validation Scope

Verify that the Temperature Target PMT register reflects correct thermal target values from three independent readout paths: 1. Register: temperature_target MMIO register (bits: FAN_TEMP_TARGET_OFFSET [15:8], REF_TEMP [23:16], TJ_MAX_TCC_OFFSET [31:24]) 2. MSR: IA32_TEMPERATURE_TARGET (MSR 0x1A2) -- same fields 3. DFX Mailbox: PCS index 0x10 (ref_temp, tj_max_tcc_offset) and MSR-read opcode (fan_temp_target_offset) All three must agree on the same values for FAN_TEMP_TARGET_OFFSET, REF_TEMP, and TJ_MAX_TCC_OFFSET.
