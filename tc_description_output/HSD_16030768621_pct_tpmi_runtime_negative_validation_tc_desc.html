PCT Micro-Architecture Context
PCT (Priority Core Turbo) = silicon-heavy feature with PCode orchestration.
HP cores (CLOS[0/1]): get elevated SST-TF turbo ratio; prefer shallower C-states.
LP cores (CLOS[2/3]): get LP turbo ratio; allowed deeper C-states.
CAPID4.bit29: fuse gate — must be 1 for PCT to be usable.
BIOS knobs: PctHpModuleCount (multiples of 4), PctCapableSystem.
NWP topology: 2 CBBs × 48 cores = 96 total; check both CBBs.

1. Validation Scope
Objective: Verify that PCode correctly rejects or handles invalid PCT partition count configurations via TPMI.
Invalid conditions tested: (1) HP count not a multiple of 4, (2) HP count < 4 (minimum partition), (3) HP count > SST_TF_INFO_1.NUM_CORE_0 (exceeds silicon max).
Expected behavior: PCode should either reject the write, clamp to valid value, or maintain last valid config. No MCA or system hang.

from pysvtools.xmlcli import nvram
import diamondrapids.pm.Active_PM.SST.pct.pct_focus as pctd
ram = nvram.getNVRAM(); ram.pull()

# Save baseline
baseline = ram.PctHpModuleCount.getValue()
max_hp = sv.socket0.cbb0.base.tpmi.sst_tf_info_1.num_core_0.get_value()
print(f'Baseline HP count: {baseline}, Max supported: {max_hp}')

def check_pct_state(label):
    pctd.generate_core_list()
    print(f'[{label}] HP cores: {len(pctd.HP_CORES)}')
    for cbb in sv.sockets.cbbs:
        en = cbb.base.tpmi.sst_cp_control.sst_cp_enable.get_value()
        print(f'  CBB{cbb.instance} SST_CP_ENABLE={en}')

# Test 1: Non-multiple of 4 (e.g., 3)
pctd.pct_bios_knob_change(cores=3)  # should be rejected (not %4)
check_pct_state('non-multiple-of-4')
# BIOS push + reboot would be needed for runtime test; use TPMI direct:
# Attempt direct TPMI write with invalid count — verify PCode ignores

# Test 2: Value less than 4
pctd.pct_bios_knob_change(cores=2)  # below minimum
check_pct_state('below-minimum')

# Test 3: Value exceeds NUM_CORE_0
pctd.pct_bios_knob_change(cores=max_hp + 4)  # exceeds silicon max
check_pct_state('exceeds-max')

# Restore valid config
pctd.pct_bios_knob_change(cores=baseline)
print(f'Restored to baseline: {baseline}')
✅ PASS: Invalid values rejected by pct_bios_knob_change (function refuses to push); PCode maintains last valid HP set; no MCA.
❌ FAIL: Invalid value accepted → PCode does not validate partition count; TPMI state corrupted. MCA or hang → PCode crash on invalid input (firmware bug).
