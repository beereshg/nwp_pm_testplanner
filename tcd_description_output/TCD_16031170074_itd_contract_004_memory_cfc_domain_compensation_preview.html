1. Architecture / Micro-architecture and Functionality

Memory/CFC Domain Compensation validates temperature-dependent voltage compensation for VccCFCMEM (memory fabric) and MLC SSA domains. These domains target the memory controller fabric and mid-level cache voltage rails, compensating for temperature-dependent threshold voltage shifts to maintain margin. > Architecture overview: See TPF — ITD Compensation §Design Details.

Key Behavioral Facts

  • FW owner: IMH Primecode
  • Fuse-programmed slope/cutoff per domain
  • Memory fabric ITD is critical during memory-intensive workloads where thermal stress is highest
  • Dual-slope may apply to memory domains — check fuse *_ITD_SLOPE_2 presence
