1. Architecture / Micro-architecture and Functionality

Feature Overview

PCT (Priority Core Turbo) is a distinct Intel PM feature that uses SST-TF’s CLOS-based frequency partitioning to designate a small subset of High Priority (HP) cores to operate at an elevated turbo frequency (up to P0max, ~4.4 GHz on NWP), while remaining Low Priority (LP) cores are clipped to approximately P1 to free the power budget for the HP cores. PCT is not an alias for SST-TF. It is tracked as its own feature (PCT HAS, IC PCT HAS) and adds: BIOS partition/core-select knobs, CLOS register programming, DLCP die-level cherry-picking, and OS discovery registers.

NWP scope: PCT is one of only 2 SST features active on NWP (SST-TF and PCT). SST-PP, SST-CP standalone, SST-BF, and HGS are ZBB’d (HSD 22021155414).

Enabling Path

BIOS CPL3
├── Check PCT capability: PCT Partition Count BIOS knob > 0 (default = 4)
│     Note: CAPID4.bit29 is GNR-only; DMR+ and NWP use Partition Count to gate PCT
├── PrimeCode (NIO reset Phase 5):
│     reads SST_TF fuses + PCT_Module_Mask → populates SST_TF_INFO_0/2/10
├── BIOS programs per-CBB TPMI (NWP: cbb0 + cbb1):
│   ├── SST_CLOS_CONFIG[0].max = SST_TF_INFO_2.RATIO_0  (HP ceiling: ~4.4 GHz)
│   ├── SST_CLOS_CONFIG[3].max = SST_TF_INFO_0.LP_CLIP_RATIO_0  (LP ceiling: ≥P1)
│   ├── SST_CP_CONTROL.SST_CP_PRIORITY_TYPE = 1  (Ordered Throttling)
│   ├── SST_CLOS_ASSOC[core]: HP cores → CLOS[0]; LP cores → CLOS[3]
│   └── SST_PP_CONTROL.feature_state[1] = 1  (SST-TF active)
└── Override MSR 0x1AD with SST_TF_INFO_2.RATIO_0  (HP TRL broadcast)

HP selection: 96 cores ÷ 4 partitions = 24 cores/partition; first 2 cores per partition = HP
              ⇒ 8 HP cores total on NWP (CCB HSD 14026595435)

Discovery Path

Interface Register / Address What It Reveals
TPMI SST_HEADER.CAPABILITY_MASK Feature availability bitmap; PCT active bit
TPMI SST_TF_INFO_0.LP_CLIP_RATIO_0 LP core frequency ceiling (≥P1, SKU-fused)
TPMI SST_TF_INFO_2.RATIO_0 HP TRL ratio (PCT turbo frequency target)
TPMI SST_TF_INFO_10 DLCP: PCT_Module_Mask — fused HP core positions per die
TPMI SST_CLOS_ASSOC_* Per-core CLOS assignment: HP=CLOS[0/1], LP=CLOS[2/3]
TPMI SST_CP_CONTROL.sst_cp_enable PCT globally enabled
MSR IA32_HWP_CAPABILITIES (0x771) highest_perf: HP cores = P0max; LP cores = LP clip
MSR 0x1AD PRIMARY_TURBO_RATIO_LIMIT HP TRL broadcast by BIOS at boot
BIOS NVRAM PctHpModuleCount Number of HP modules configured
