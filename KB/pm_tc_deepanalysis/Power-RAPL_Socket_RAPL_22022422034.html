Test Intent

Verify Socket RAPL PL1/PL2 power limiting and Tau time window programming on NWP, validating the NN-PID closed-loop controller converges to target power limits within spec timing. Validate TPMI register programming, Lock bit functionality, Tau clipping, and power convergence via ENERGY_STATUS monitoring.

Pre-Conditions

  • Platform booted to XOS/SVOS
  • PythonSV environment active with namednodes access
  • Socket RAPL functional (TPMI_ID=0x00 accessible)
  • BIOS has not locked RAPL registers (or test runs pre-lock)

Test Steps

Step 1 — Read POWER_UNIT to get scaling factors (TPMI offset 0x08)
# NWP uses nio (not imh0/imh1) unit_reg = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_power_unit.read() pwr_unit = unit_reg & 0xF # [3:0] PWR_UNIT → 2^(-pwr_unit) W energy_unit = (unit_reg >> 6) & 0x1F # [10:6] ENERGY_UNIT → 2^(-energy_unit) J time_unit = (unit_reg >> 12) & 0xF # [15:12] TIME_UNIT → 2^(-time_unit) s print(f"PWR_UNIT={pwr_unit}, ENERGY_UNIT={energy_unit}, TIME_UNIT={time_unit}")

Verification: PWR_UNIT typically = 3 (0.125W resolution), ENERGY_UNIT = 14, TIME_UNIT = 10

Step 2 — Read PL_INFO for TDP defaults (TPMI offset 0x48)
pl_info = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_pl_info.read() max_pl1 = pl_info & 0x3FFFF # [17:0] MAX_PL1 = TDP (in PWR_UNIT) min_pl = (pl_info >> 18) & 0x3FFFF # [35:18] MIN_PL max_pl2 = (pl_info >> 36) & 0x3FFFF # [53:36] MAX_PL2 = 1.2 × TDP tdp_watts = max_pl1 * (2 ** -pwr_unit) print(f"TDP={tdp_watts}W, MAX_PL2={max_pl2 * (2 ** -pwr_unit)}W")

Verification: MAX_PL1 matches fused TDP; MAX_PL2 ≈ 1.2 × TDP

Step 3 — Write PL1_CONTROL with test power limit (TPMI offset 0x10)
# Set PL1 to 70% TDP for throttle test test_pl1 = int(max_pl1 * 0.70) pl1_ctrl = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_pl1_control current = pl1_ctrl.read() # Clear PWR_LIM [17:0], set new value, enable [62] new_val = (current & ~0x3FFFF) | test_pl1 | (1 << 62) pl1_ctrl.write(new_val) print(f"PL1 set to {test_pl1 * (2 ** -pwr_unit)}W (70% TDP)")

Verification: Read back confirms PWR_LIM [17:0] = test_pl1, PWR_LIM_EN [62] = 1

Step 4 — Test LOCK bit prevents modification (TPMI offset 0x10)
# Set LOCK [63] = 1 pl1_ctrl = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_pl1_control locked_val = pl1_ctrl.read() | (1 << 63) pl1_ctrl.write(locked_val) # Attempt to change PWR_LIM — should be rejected pl1_ctrl.write(locked_val ^ 0x100) # flip a bit in PWR_LIM readback = pl1_ctrl.read() assert readback == locked_val, "FAIL: Lock bit bypassed!" print("PASS: Lock bit prevents writes")

Verification: Write after lock has no effect; register value unchanged

Step 5 — Sweep PL1 TIME_WINDOW (Tau) and verify clipping (TPMI offset 0x10)
# PL1 Tau valid range: 1s - 5s (TIME_WINDOW [24:18]) # Encoding: Y × 2^Z where Y = bits [22:18], Z = bits [24:23] pl1_ctrl = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_pl1_control def encode_tau(seconds, time_unit): # Convert seconds to TIME_WINDOW encoding raw_time = seconds / (2 ** -time_unit) # Find best Y, Z fit return int(raw_time) & 0x7F # simplified for tau_s in [0.5, 1.0, 2.0, 5.0, 10.0]: encoded = encode_tau(tau_s, time_unit) # Write and read back print(f"Tau={tau_s}s → encoded={encoded}, clipped to valid range")

Verification: Values < 1s clipped to 1s; values > 5s clipped to 5s

Step 6 — Sweep PL2 TIME_WINDOW (TPMI offset 0x18)
# PL2 Tau valid range: 11.7ms - 39ms pl2_ctrl = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_pl2_control # Similar sweep as PL1 but with ms-range values for tau_ms in [5, 11.7, 25, 39, 50]: print(f"PL2 Tau={tau_ms}ms test")

Verification: Values clipped to [11.7ms, 39ms] range

Step 7 — Apply sustained workload
# Use pmutil or stress-ng to load all cores # This drives power above PL1 to trigger throttling import subprocess subprocess.Popen(["stress-ng", "--cpu", "96", "--timeout", "60s"])

Verification: All 96 cores loaded; power exceeds PL1 target initially

Step 8 — Monitor ENERGY_STATUS and calculate power (TPMI offset 0x38)
import time energy_reg = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_energy_status samples = [] for i in range(50): raw = energy_reg.read() energy = raw & 0xFFFFFFFF # [31:0] ENERGY counter timestamp = (raw >> 32) & 0xFFFFFFFF # [63:32] TIME (10ns units) samples.append((energy, timestamp)) time.sleep(0.1) # 100ms interval # Calculate power: P = ΔE / Δt for i in range(1, len(samples)): dE = (samples[i][0] - samples[i-1][0]) * (2 ** -energy_unit) # Joules dt = (samples[i][1] - samples[i-1][1]) * 10e-9 # seconds power_w = dE / dt print(f"Sample {i}: Power = {power_w:.1f}W")

Verification: Power readings show convergence toward PL1 target over time

Step 9 — Verify power converges within 3-5× Tau
# After 3-5× Tau, power should be within ±5% of PL1 tau_s = 2.0 # example Tau setting wait_time = tau_s * 5 time.sleep(wait_time) final_power = # calculate from last samples pl1_target = test_pl1 * (2 ** -pwr_unit) tolerance = pl1_target * 0.05 assert abs(final_power - pl1_target) < tolerance, \ f"FAIL: Power {final_power}W not within 5% of {pl1_target}W" print(f"PASS: Power converged to {final_power}W (target {pl1_target}W)")

Verification: Power within ±5% of PL1 after convergence time

Step 10 — Check PERF_STATUS throttle counter (TPMI offset 0x40)
perf_status = sv.socket0.nio.punit.ptpcfsms.ptpcfsms.socket_rapl_perf_status.read() throttle_ctr = perf_status & 0xFFFFFFFF # [31:0] PWR_LIMIT_THROTTLE_CTR assert throttle_ctr > 0, "FAIL: Throttle counter not incrementing" print(f"PASS: Throttle counter = {throttle_ctr}")

Verification: Counter increments during RAPL-limited operation

Step 11 — Verify MSR deprecation (MSR 0x610, 0x611)
# On NWP, legacy MSRs are deprecated and return 0 msr_610 = sv.socket0.core0.msr(0x610) # IA32_PKG_RAPL_LIMIT msr_611 = sv.socket0.core0.msr(0x611) # IA32_PKG_ENERGY_STATUS assert msr_610 == 0, f"FAIL: MSR 0x610 = {hex(msr_610)}, expected 0" assert msr_611 == 0, f"FAIL: MSR 0x611 = {hex(msr_611)}, expected 0" print("PASS: Legacy MSRs deprecated (return 0)")

Verification: MSR reads return 0; TPMI is the only valid interface
