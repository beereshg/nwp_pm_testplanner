Refined Intent

Sweep BIOS-to-Pcode (PCU_CR_BIOS_MAILBOX at 0x705c/0x7058) and OS-to-Pcode (MSR 0xB0/0xB1) mailbox commands. Verify valid responses and side-effects for each command code.

Refined Test Steps

Pre-Conditions
  • Platform booted, BIOS and Pcode initialized
  • Mailbox interfaces accessible via PythonSV
Execution Steps

Step 1 — BIOS mailbox sweep:

For each command in [0xA6 SET_MC_FREQ, 0xA7 READ_MC_FREQ, ...]:

a. Write command to PCU_CR_BIOS_MAILBOX_INTERFACE (0x705c)

b. Write data to PCU_CR_BIOS_MAILBOX_DATA (0x7058)

c. Set run_busy, wait for completion

d. Read return code — verify no error

e. Read data register — verify expected response

Step 2 — OS mailbox sweep:

For each OS mailbox command:

a. Write MSR 0xB1 (OS_MAILBOX_DATA)

b. Write MSR 0xB0 (OS_MAILBOX_INTERFACE) with command + run_busy

c. Wait for run_busy to clear

d. Read MSR 0xB0 return code — verify no error

Step 3 — Verify side-effects:

After SET commands, read related registers to confirm the setting took effect.

After READ commands, verify returned data matches expected register values.

Verification

PASS: All supported commands return success, data matches expected

FAIL: Mailbox timeout, error return code, or data mismatch

def os_mailbox(socket_number=0, imh=0, command=0, sub_command=0, parameter=0, data=None):
    """ Sends an OS to Pcode Mailbox command and returns the response """

    socket = namednodes.sv.socket.getBySocket(socket_number)

    os_mailbox_interface = socket.getbypath(f"imh{imh}").punit.ptpcfsms.ptpcfsms.os_mailbox_interface
    os_mailbox_data = socket.getbypath(f"imh{imh}").punit.ptpcfsms.ptpcfsms.os_mailbox_data

    wait(os_mailbox_interface)

    print(f"PM_INFO :: Writing PLR Mailbox Interface to 0x{command + (sub_command<<8) + (parameter << 16) + (1 << 31):x}")

    # If there's data, write it first to be consumed when we start the mailbox command
    if data is None:
        os_mailbox_data.write(0)
    else:
        print(f"PM_INFO :: Writing PLR Mailbox Data to 0x{data:x}")
        os_mailbox_data.write(data)
    os_mailbox_interface.command.store(command)
    os_mailbox_interface.sub_command.store(sub_command)
    os_mailbox_interface.parameter.store(parameter)
    os_mailbox_interface.run_busy.store(1)
    os_mailbox_interface.flush()

    wait(os_mailbox_interface)

    return_code = os_mailbox_interface.command.read()
    return_data = os_mailbox_data.read()

    print(f"PM_INFO :: OSMB return Code: 0x{return_code:x}")
    print(f"PM_INFO :: OSMB return Data: 0x{return_data:x}")


    return return_data, _MAILBOX_ERRORS[return_code]


def wait(interface = None):
    """ Inject wait-for-cycle commands into emulation """
    wait_time = 0
    wait_cycles = 100000
    wait_timeout = 20

    while interface.run_busy.read():
        if ACCESS_METHOD == "stub":
            interface.write(0)
        elif ACCESS_METHOD == "emulation":
            cli.run_command(f"emu.engine.wait-for-cycle -relative {wait_cycles}")
        elif ACCESS_METHOD == "simics":
            cli.run_command(f"run-cycles {wait_cycles}")
        else:
            time.sleep(1)

        wait_time += 1
        wait_cycles += 10000

        if wait_time > wait_timeout:
            return -1
    return 0
