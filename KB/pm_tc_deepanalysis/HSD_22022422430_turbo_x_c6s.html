Section A: NWP Disposition & Justification

**Disposition: Runnable_On_N-1**

This test verifies **Turbo × C6S** cross-product — interaction between Turbo Mode and C6S (C6 with LLC Save — deeper than C6A). In C6S, core powers off and the LLC associated with that core is flushed/saved, enabling deeper power reduction.

C6S is a core-level idle state functional on NWP. Not to be confused with Package C6 or Ring C6/MC6 (both ZBB on NWP).

**Key Justification:**

  • Core C6S functional on NWP (not ZBB)
  • `DMR_PO` + `plc.ti_gate.b0` + `plc.feature.p1` + `pm.xproducts.pm` tags
  • C6S provides deeper idle than C6A (LLC flush) → more Turbo headroom
