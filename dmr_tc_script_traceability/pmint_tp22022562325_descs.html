env: silicon
Validation Scope

HAS Ref — PMX CPU/Core Power × Traffic Interaction: PMX cross-product with active CPU traffic validates that PM state transitions (C-states, P-states, DVFS) occur correctly under load. Key: core power draw with PMX-sourced traffic must not exceed per-core power limits; frequency and voltage transitions complete without glitch; traffic throughput degrades gracefully under PM throttling.

3Collect PM registers, RAPL counters, and NLOG after test.All values within spec; system stable; NLOG error-free.

22022421695 NEW pmx Security x PC6

Validation Scope

HAS Ref — PMX Security × PC6 Interaction: Security flows (TDX, SGX, TME, MKTME) must not interfere with PC6 entry/exit.
